# Agent Concierge

A prototype of what bespoke, agent-driven experiential travel planning feels like in 2026.

> World-class README follows once the prototype lands. For now, see `DESIGN.md` for design notes and `.claude/plans/build-a-digital-agentic-ethereal-cray.md` for the architecture plan.

## Dev

```sh
npm install
npm run dev
```

## Modes

- **Mock** (default): scripted end-to-end demo, no network.
- **Live**: set `VITE_GEMINI_API_KEY` in `.env.local` and switch Live in the settings sheet.
